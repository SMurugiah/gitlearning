Git Learning


Git Basic Commands

    1. $ git init
    2. $ git add
    3. $ git status
    4. $ git commit
    5. $ git push
    6. $ git pull
    7. $ git clone

    2.1 $ git add Filename.extn or Filename.* or . (add all files)

Other Commands

    1. $ git --version
    2. $ touch filename.extn //(Create New File)

Adding User Name & Email Address

    $ git config --global user.name 'Sathish Murugiah'
    $ git config --global user.email 'Sathish.Murugiah@gmail.com'

Commit Files (Two Types)

    1. $ git commit //(this will open editor but nothing editable unless press 'i' Key and type command 'Initial Commit' press 'Escape' key and type :wq to come out)
    2. $ git commit -m 'User Comments' // (this -m will skip the editing part)

Git Ignore

$ touch .gitingnore (once gitignore created, add filename to .gitignore file same applicable to directories as well) example: adding file to gitignore (filename.extn or *.extn) adding directories (/dir name)

Branches

    $ git branch branchname //Create Branch
    $ git checkout branchname //switch to start working in the branch
    Note: Once we switch back to master from branch none of the branch files will be seen, will be visible once merge done

MErge MAster and branch

    $ git merge branchname (this will open editor)

Adding Local Repo in Remote Repo

    1. $ git remote
    2. git remote add origin https://github.com/SMurugiah/gitlearning.git Create Remote Repo (Once created u will get command like above with remote repo details copy the command)
    3. git push -u origin master (push local repo to remote repo) Refresh remote repo to see all files

Push Files from local repo to remote repo

    1. $ git push

Pull

    $ git pull

Clone

    $ git clone https://github.com/SMurugiah/gitlearning.git Create a loca repo folder, open bash and git clone link (copied from remote repo)
